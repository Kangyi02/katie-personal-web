# Katie Zhang — Personal Portfolio

A warm, personal portfolio website built with React + Vite.

## Tech Stack

- **React 18** — UI framework
- **Vite** — Build tool & dev server
- **CSS Modules** — Component-scoped styling
- **Google Fonts** — Lora (serif) + DM Sans (sans-serif)

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Adding Your Profile Photo

Place your profile photo at:
```
public/profile.jpg
```

The site will automatically display it in the hero section. If no photo is found, it falls back to the "KZ" initials placeholder.

## Sections

- **Hero** — Name, tagline, social links
- **About** — Bio, education, awards
- **Skills** — Languages, frameworks, tools, research
- **Experience** — Timeline of work & volunteer roles
- **Projects** — Highlighted projects with tech tags
- **Blog** — Coming soon previews
- **Contact** — All contact channels

## Deployment

This project is ready to deploy on [Vercel](https://vercel.com). Just push to GitHub and import the repo in Vercel — no configuration needed.
